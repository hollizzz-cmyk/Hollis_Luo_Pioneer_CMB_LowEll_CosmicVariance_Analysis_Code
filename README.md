# Investigating Low-ℓ Power Suppression in the Planck 2018 CMB Temperature Spectrum Using Cosmic-Variance Analysis

This repository contains the computational pipeline developed for the Pioneer Academics research project:

**"Investigating Low-ℓ Power Suppression in the Planck 2018 CMB Temperature Spectrum Using Cosmic-Variance Analysis."**

The project studies the observed suppression of large-scale temperature fluctuations in the Cosmic Microwave Background (CMB) and evaluates whether the deviation from the standard ΛCDM prediction represents a statistically significant anomaly or a fluctuation consistent with cosmic variance.

---

## Research Overview

The low multipole (low-ℓ) region of the CMB temperature power spectrum contains information about the largest observable scales of the Universe. Previous studies have reported reduced power at low multipoles compared with ΛCDM predictions.

This project investigates this feature by:

- Generating theoretical ΛCDM CMB temperature spectra using **CAMB**
- Processing and comparing against **Planck 2018 TT power spectrum data**
- Calculating observational residuals and fractional differences
- Evaluating deviations using cosmic variance uncertainties
- Performing statistical significance analysis
- Testing phenomenological primordial power spectrum cutoff models
- Using Monte Carlo simulations to evaluate the likelihood of observing similar low-ℓ deviations
